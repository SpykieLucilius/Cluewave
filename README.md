# Current features (for teacher)

## Authentication and User management

- User registration and Login
- Account persistence
- Input Validation and DTOs
- JWT issuance and verification
- Security filter and configuration
- Password encoding
- Edit profile with password required
- Added Login with Google (Not activated cause it require a payment of 40.- so it miss the CLIENT_ID)
## Room

- Room creation with host email
- Room limitation to 2 player max
- Joining room with room code
- Joining room with host email
- Restrictions (unable to join unexisting room, full room and create two room with same code)
## Database

- Database persistance with Cloud DB (Supabase)
- Pooling

## Deployment

- Deployed on Heroku
- Linked to a Domain 
- Use of SSL Certificate
- Use of Configuration Variable for secret security
